NOTA: Si tiene instalada una versión previa, deberá desinstalar esta en la opción PANEL DE Control / instalar o Quitar Programas.

 

Una vez insertado el disco compacto se deberá de presionar el botón INICIO de la ventana principal de Windows, posteriormente la opción EJECUTAR y con el botón EXAMINAR deberá de seleccionar el archivo SETUP.EXE el cual se encuentra en el disco compacto de Instalación.

 

Se deberán de seguir las instrucciones del proceso de instalación, pudiendo seleccionar el grupo de programas y la carpeta donde se desea que se instale el sistema, por default esta se instala en \LACTOW30.

 

La llave del sistema se instalara en el puerto (PARALELO o USB) dependiendo del modelo con que se cuente.

Es importante la instalación de este dispositivo,  ya que sin este el sistema no funcionara.

 

Una vez instalado el dispositivo se deberá de acceder la carpeta Drivers del CD de Instalación y ejecutar el archivo INSTDRV.EXE el cual instalara los drivers o manejadores correspondientes para este dispositivo.

 

Es importante la instalación de estos drivers sino el sistema no funcionara correctamente.

 

Una vez instalado el sistema y los drivers, se seleccionara del menú de programas de Windows programas la opción Lactofox Software Lechero - Lactofox 3.0 y una vez iniciado se deberá de seleccionar la opción REGISTRO Y CLAVES DE ACCESO del Menú Principal, donde se deberá de proporcionar el numero de REGISTRO O ACTIVACION del Sistema, este deberá de ser proporcionado por su distribuidor, así como si se adquirieron otros módulos, se deberán de proporcionar las claves correspondientes.

 

En caso de no contar con numero de registro, este podrá ser solicitado a AGROPLUS SA de CV proporcionando el numero de serie, así como fecha de caducidad, numero de hatos y vientres que aparezcan en la ventana de REGISTRO.

 

Posteriormente se deberá de seleccionar la opción DIRECTORIO DE HATOS donde se dará(n) de alta los hatos, creándose de esta forma los archivos necesarios para la captura y proceso de la información.

 

En caso de que la opción de ayuda no funcione correctamente se deberá de ejecutar el archivo HHUPD.EXE del directorio de instalación, el cual instala y registra los componentes necesarios.

 

El Explorador de Internet o el motor de ejecución del Explorador de Internet tienen que estar previamente instalados.
